| 명령어 | 한 줄 요약 | 중요도 |
| :--- | :--- | :---: |
| **top** | 실시간 시스템 모니터링 | ⭐⭐⭐ |
| **ps** | 현재 실행 중인 프로세스 확인 | ⭐⭐⭐ |
| **jobs** | 백그라운드 작업 목록 확인 | ⭐⭐ |
| **kill** | 프로세스 강제 종료 | ⭐⭐⭐ |

---

*   ### 📊 top
    *   **설명**: 컴퓨터의 CPU와 메모리 사용량을 실시간으로 보여줍니다.
    *   **팁**: `q`를 누르면 화면에서 빠져나올 수 있어요!

*   ### 📝 ps
    *   **설명**: 현재 어떤 프로그램들이 켜져 있는지 스냅샷을 찍듯 보여줍니다.
    *   **자주 쓰는 옵션**: `ps -ef` 또는 `ps aux`

*   ### ⏳ jobs
    *   **설명**: 현재 터미널 세션에서 뒤로 돌린(백그라운드) 작업들의 상태를 보여줍니다.

*   ### 🛑 kill
    *   **설명**: 멈춘 프로그램을 종료할 때 사용합니다.
    *   **예시**: `kill -9 [프로세스 ID]` (강제 종료)
